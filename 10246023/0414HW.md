4.請寫一段 Ruby 程式，讓 `puts "hello world".count_character` 可以在畫面上印出 `10`
```
class string
  def count_character
    a = self.delete(" ").split("")
    return a.size
  end
end
puts ("puts "hello world").count_character
```
