1.請完成以下實作練習：

```
class Animal
  def sleep
      puts "ZZZZZZ"
  end
  def  eat
      puts "Yammy!"
  end
end

class Cat < Animal; end
class Dog < Animal; end

kitty = Cat.new

lucky = Dog.new
lucky.eat       # => "Yammy!"
```

2.請完成以下實作練習：

```
class Cat
  attr_accessor :age
  def initialize (age)
    @age = age
  end
end

kitty = Cat.new(10)
puts kitty.age       # => 10
kitty.age = 18
puts kitty.age       # => 18
```

4.請寫一段 Ruby 程式，讓 `puts "hello world".count_character` 可以在畫面上印出 `10`

```

  def count_character
    self.length - self.count(" ")
  end
end
```
