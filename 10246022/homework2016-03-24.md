* 請寫一段 Ruby 程式，把 `[1, 3, 5, 7, 9]` 變成 `[2, 4, 6, 8, 10]`
```
a = [1, 3, 5, 7, 9]
p a.map{|num| num+1}
```
* 請印出在 1 到 2000 中，可以被 4 整除，但不能 100 整除，但又要可以被 400 整除的數字。
```
p (1..2000).to_a.select{|num| num % 4 ==0 && num % 100 !=0 && num % 400 ==0}
```
* 請印出在 1 到 100 中所有單數的總和。
```
a = (1..100).to_a.select {|num| num.odd?}
p a.inject(:+)
```
